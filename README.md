# AI 实习岗位分析助手

一个面向求职场景的 AI 岗位分析工具。用户可以粘贴实习岗位 JD，系统会自动提取岗位核心关键词，结合个人技能画像计算岗位匹配度，并生成简历优化建议、投递建议和 BOSS 打招呼话术。
## 在线预览
https://zelin-z.github.io/ai-job-agent/
## 技术栈

- Vue 3
- TypeScript
- Element Plus
- Vite
- localStorage

## 核心功能

### 1. JD 关键词提取

基于岗位描述内容，自动识别 Vue3、TypeScript、Axios、Element Plus、MySQL、Java、Spring Boot、CRM、ERP 等关键词。

### 2. 岗位匹配度计算

根据岗位关键词与个人技能画像的匹配情况，动态计算岗位匹配度，并通过进度条展示。

### 3. 技能差距分析

自动识别岗位中用户暂未掌握的技能，例如 Java、Spring Boot，并生成待补充技能标签。

### 4. 简历优化建议

根据岗位匹配情况，动态生成简历优化建议，帮助用户突出已匹配技能，避免过度包装不熟悉的技术。

### 5. 投递建议生成

根据匹配度和缺失技能，给出“建议投递 / 谨慎投递 / 暂不建议优先投递”的判断。

### 6. BOSS 打招呼话术生成

根据岗位方向和关键词，自动生成定制化打招呼话术，并支持一键复制。

### 7. 历史记录本地保存

使用 localStorage 保存历史分析记录，刷新页面后记录仍然保留，并支持一键清空。

## 项目运行

```bash
npm install
npm run dev
```
